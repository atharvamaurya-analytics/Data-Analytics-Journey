# Excel Basics

## 1. What is Excel?

Microsoft Excel is a spreadsheet application used to store, organize, calculate, analyze, and visualize data.

Data analysts use Excel for:
- Data cleaning
- Data analysis
- Report creation
- Dashboard creation
- Business insights

---

## 2. What is a Workbook?

A workbook is an Excel file that contains one or more worksheets.

Example:

Sales_Report.xlsx

A workbook can contain multiple sheets like:

- January Sales
- February Sales
- Summary Report

---

## 3. What is a Worksheet?

A worksheet is an individual page inside an Excel workbook where data is stored and managed.

Example:

Workbook:
Sales_Report.xlsx

Worksheets:
- Sales Data
- Customer Data
- Summary

---

## 4. Rows and Columns

### Rows

Rows are horizontal lines in Excel.

Rows are identified using numbers:

1, 2, 3, 4...

### Columns

Columns are vertical lines in Excel.

Columns are identified using letters:

A, B, C, D...

---

## 5. What is a Cell?

A cell is the intersection point of a row and a column.

Example:

A1

A = Column  
1 = Row

Cells can contain:
- Text
- Numbers
- Dates
- Formulas

---

## 6. Data Types in Excel

### Text

Examples:
- Product Name
- Customer Name

### Numbers

Examples:
- 500
- 1000
- 2500

### Dates

Example:

15-06-2026

### Formulas

Example:

=SUM(A1:A10)

---

## 7. What are Excel Formulas?

Formulas are instructions used to perform calculations in Excel.

Every formula starts with:

=

Example:

=A1+B1

This adds the values of A1 and B1.

---

# Basic Excel Functions

## SUM()

Used to add values.

Example:

=SUM(A1:A10)


## AVERAGE()

Used to calculate average.

Example:

=AVERAGE(A1:A10)


## COUNT()

Counts numeric values.

Example:

=COUNT(A1:A10)


## MAX()

Returns the highest value.

Example:

=MAX(A1:A10)


## MIN()

Returns the lowest value.

Example:

=MIN(A1:A10)

---

# Sorting

Sorting is used to arrange data in a specific order.

Example:

Before:

500  
100  
900


After sorting:

100  
500  
900

---

# Filtering

Filtering is used to display only required data.

Example:

From thousands of sales records, we can filter only:

Delhi Sales Data

---

# Data Cleaning

Data cleaning means preparing raw data for analysis.

Common cleaning tasks:

- Removing duplicates
- Handling missing values
- Correcting spelling mistakes
- Removing extra spaces

---

# Pivot Table

Pivot tables are used to summarize large datasets.

Examples:

- Total sales by city
- Sales by product
- Monthly reports

---

# Charts

Charts represent data visually.

Common Excel charts:

- Bar Chart
- Line Chart
- Pie Chart
- Column Chart

---

# Importance of Excel for Data Analysts

Excel is an important tool for data analysts because it helps in:

- Data cleaning
- Data analysis
- Reporting
- Dashboard creation
- Finding business insights
